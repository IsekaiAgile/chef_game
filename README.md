# ネコノヒゲ亭 (Agile Kitchen Game) 🐾🍳

アジャイルの原則をキッチン経営を通して学ぶ、ブラウザベースのシミュレーション・ビジュアルノベルゲームです。

## 🎮 プレイ方法 (How to Play)

### ストーリー
あなたは「ネコノヒゲ亭」の新人見習いシェフ。
最初の7日間（Episode 1）は、正式採用をかけた「試用期間（7-Day Sprint）」です。
7日目の審判（Judgment Day）までに**成長値 (Growth) 50以上**を達成し、無事に採用を勝ち取りましょう！

### ゲームの流れ
毎日、以下の3つのフェーズを繰り返します。

1. **Morning Stand-up（朝会）**
   - その日の重点項目（Quality / Speed / Experiment）を選択します。
   - 選択によって、アクション時のボーナスやリスクが変化します。

2. **Action Phase（行動フェーズ）**
   - 1日3回のアクションを選択します。
     - **調理 (Cook):** 料理を出し、経験を積みます。
     - **分析 (Analyze):** プロセスを見直し、技術負債を解消します（成功率向上）。
     - **対話 (Talk):** 仲間と対話し、チームの調子を整えます。

3. **Night Retrospective（ふりかえり）**
   - 1日の成果を確認します。
   - **Pivot（方向転換）**を選択して、翌日の戦略を練り直すことも可能です。

### 特殊イベント：スパイス危機 (Spice Crisis)
- **Day 3 〜 Day 4**に発生。
- 調理の難易度が上がりますが、ここを乗り越えることが成長の鍵となります。

### 勝利条件
- **Day 7** 終了時の判定シーンで、**Growth（成長値）が50以上**であること。

## 🛠 技術スタック
- **Language:** Pure JavaScript (ES6+)
- **Architecture:** SOLID原則に基づいた疎結合設計
- **System:** EventBusによるイベント駆動型
- **UI:** CSS Grid/Flexboxを用いたレスポンシブデザイン（モバイル対応）
